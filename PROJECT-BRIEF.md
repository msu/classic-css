# classic.css

classic CSS is going to be the basis for the next version of 491/431, Hypermedia Systems.

The goals of classic.css are the following:

* Style semantic HTML reasonably well
* Provide useful basic components and use ARIA attributes to drive additional functionality (may require a paired .js file)
* Implement a neo-classical digital look and feel (see https://fixiproject.org) for button look and general inspiration
* Implement traditional usability (e.g. using depth as a visual easement)
